### Description (4000 chars max)

TinyText is a clean, native plain text editor for macOS. Open a file, edit it, move on. No projects, no tabs you didn't ask for, no startup delay. Just a fast, focused window on your text.

Features:
- Opens any plain text file — .txt, .text, .cfg, .conf, .ini, .config, .properties, .env, .rc, and more
- Folder sidebar with directory browsing and favorites
- Multi-tab editing with auto-save and dirty state tracking
- Native Find & Replace (Cmd+F)
- Quick open file finder (Cmd+P)
- Adjustable font size (Cmd+/Cmd-)
- Word wrap toggle (Option+Z)
- Line numbers toggle (Option+L)
- Smart text pairs — auto-close brackets, quotes, and parentheses
- Line manipulation — move, duplicate, or delete lines with keyboard shortcuts
- File watcher — reloads automatically when files change externally
- Multiple independent windows
- Export to PDF or HTML
- Light and dark mode — follows your system
- Opens files directly from Finder
- On-device AI assistant (Cmd+K) — edit, transform, and ask questions about your text. Runs via CoreML, no API key required

The missing base layer of TinySuite. Every structured format — JSON, CSV, Markdown, SQL — has its own Tiny app. TinyText handles everything else. It's the app you want instead of TextEdit.

Built entirely with native macOS technologies. No Electron. No web views. Just SwiftUI and AppKit doing what they do best.

Part of TinySuite — a family of native macOS micro-tools that each do one thing well. Every app is built around the file, not around itself. No accounts, no cloud, no lock-in. Open your file, understand it, act on it. Learn more at tinysuite.app.
